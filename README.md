# reactor - simple design prototyping tool

Simple and easy to use prototyping tool

* create prototypes
* create diagrams
* create mockups

## Installation

1. Clone the repo:

```javascript
$ git clone https://github.com/jtasek/reactor.git
```

2. Install npm packages

```javascript
$ npm install
```

OR

```javascript
$ yarn
```

3. Run the application

```javascript
$ npm start
```

OR

```javascript
$ yarn start
```
